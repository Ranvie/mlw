Lista de elementos ordenados

a lista ordenada é definida pela tag <ol>, e os elementos que são inseridos dentro da lista são definidos por <li>.

É possível modificar o numerador e o fazer mostrar algarismos romanos minúsculos e maiúsculos, letras alfabéticas maiúscula e minúsculas e valores númericos, utilizando o type. Também é possível alterar o valor inicial utilizando start. Dentre outros atributos.

  1. Banana
  2. Maçã
  3. Laranja
  4. Pêra

Lista de elementos desordenados

Possui uma estrutura parecida com a lista ordenada. Para ser definida utiliza-se a tag <ul> e seus elementos usando a tag <li>.

Com o atributo type também é possível definir o marcador da lista, podemos utilizar nos formatos circular, quadrado e alguns outros.

  • Carro
  • Bicicleta
  • Moto
  • Patinete
  • Skate

Lista de termos

Por fim, a lista de termos pode ser utilizada usando a tag <dl>, seu "título" será a tag <dt> e os elementos serão a tag <dd>.
